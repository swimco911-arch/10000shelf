# 📚 나의 제미나이 스토리북 서재 (Gemini Storybook Library)

> "제미나이가 들려주는 200가지 이야기, 이제 손가락 클릭 한 번으로 떠나요!"

## 🌟 프로젝트 소개
이 프로젝트는 제미나이(Gemini)와 함께 만든 **200권의 소중한 스토리북**들을 한눈에 보고 관리하기 위한 **가상 서재**입니다. 
단순한 링크 목록에서 벗어나, 나무 책장에 꽂힌 알록달록한 책들을 넘겨보는 즐거움을 구현합니다.

## 🛠 주요 기능
- **디지털 책장:** 장르별로 색상이 다른 책들이 나무 선반 위에 예쁘게 진열됩니다.
- **퀵 프리뷰:** 책을 누르면 표지와 AI가 작성한 짧은 요약을 팝업으로 확인할 수 있습니다.
- **스마트 리디렉션:** 마음에 드는 책을 한 번 더 누르면 바로 제미나이 스토리북으로 이동합니다.
- **자동 업데이트:** 구글 스프레드시트에 링크만 추가하면 서재에 새 책이 자동으로 꽂힙니다. (노가다 끝!)

## 🎨 서재 분류 규칙 (계획)
- 🧚 **판타지**: 보라색 책등
- 🚀 **SF/미래**: 파란색 책등
- 🍳 **요리/일상**: 초록색 책등
- 🔍 **미스터리**: 검정색 책등
- 🌊 **어드벤처**: 하늘색 책등
